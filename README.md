## How to Host on GitHub Pages

1. Go to https://github.com and sign in (or create a free account)
2. Click **"New repository"** (green button)
3. Name it: `siddhidata-dental` (or anything you like)
4. Set it to **Public**
5. Click **"Create repository"**
6. Click **"uploading an existing file"** link
7. Drag and drop **all files and the images folder** into the upload area
8. Click **"Commit changes"**
9. Go to **Settings → Pages**
10. Under "Source", select **main branch** and click Save
11. Your site will be live at: `https://yourusername.github.io/siddhidata-dental`

## To Update the Google Map

Replace the iframe `src` in `index.html` with the actual embed link:
1. Search "Siddhidata Dental Clinic Domjur" on Google Maps
2. Click Share → Embed a map → Copy the HTML
3. Paste it replacing the existing `<iframe>` in the contact section
